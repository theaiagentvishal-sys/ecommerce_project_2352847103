üîç Troubleshooting

Issue: Images Still Not Showing

Solution:

  • Check browser console (F12) for image load errors
  • Verify Unsplash URLs are accessible (try opening URL directly)
  • Clear browser cache: Ctrl+Shift+Delete
  • Hard refresh page: Ctrl+F5
  • Check if placeholder fallback appears instead

Issue: Add-to-Cart Doesn't Redirect to Login

Solution:

  • Check DevTools Console for JavaScript errors
  • Verify localStorage is cleared: localStorage.clear() in console
  • Check if UserSession.isLoggedIn() returns false when logged out
  • Verify shopping-flow.js is loaded: Search for "shopping-flow.js" in Sources tab

Issue: Cart Page Shows "Login/Register" Instead of User Name

Solution:

  • Verify you're actually logged in (check localStorage: localStorage.getItem('username'))
  • Check that updateUserStatus() function is being called in cart.html
  • Refresh page to trigger initialization
  • Check browser console for errors in updateUserStatus() function
